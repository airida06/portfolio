Overview

This project was initiated because the client wanted to redesign their website. Initially, it was a simple company profile website, but now the client wants a website that not only showcases the company profile but also includes e-commerce features.

📱 Key Features

  1. 1. Company Profile: Displays the company’s profile, services, and other essential information.
  2. 2. E-Commerce: Features product catalog, shopping cart, payment, and order tracking.

🎯 Project Direction

The design is made intuitive by considering users’ age, digital literacy, and their habits in using e-learning platforms:

  1. 1. Simple navigation
  2. 2. Clean & readable interface
  3. 3. Functional & efficient
  4. 4. Design consistency

My Role

  • Designing UI components using auto-layout, variables, variant properties, text, and more.

  • Conduct design validation with users to ensure the solution meets their needs.

  • Develop a structured design system including typography, color, and other elements.

  • Create application interfaces and hand them off to developers for implementation.

Project Flow

👩‍💻 Project Scope and Scale

  • 01. Navigation Menu

    This website allows users to view the company profile while also accessing e-commerce features, such as browsing and purchasing products.

  • 02. Three Different User Roles Views

    To meet diverse operational needs and ensure proper access control:

    1. 1. Admin: Manage the website, company profile content, and e-commerce products.
    2. 2. User: Browse company profile, view products, and make purchases.

    • 🔍 Benchmarking

    Benchmarking was conducted on competitor websites in the same industry to understand business flow models. This helps in designing a website that aligns with business needs and user expectations.

    ⚛️ Design System

    Simple Design System and Style Guide to facilitate programmers in implementing consistent UI components, colors, typography, and interactions across the website.
