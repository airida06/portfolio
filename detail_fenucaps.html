Fenucaps Academy

Overview

This project focuses on developing an e-learning feature for the Fenucaps website, which previously functioned solely as an e-commerce platform. I designed an integrated learning system inspired by platforms like Udemy, MySkill and dicoding enabling breastfeeding mothers not only to purchase products but also to easily access relevant and well-structured educational content.

📱 Key Features

  1. 1. Course Purchase & Certification: users purchase courses, access learning materials, and receive a certificate upon completion
  2. 2. Community & Discussion Forum: users can ask questions, discuss, and share experiences with other learners

🎯 Project Direction

The design is made intuitive by considering users’ age, digital literacy, and their habits in using e-learning platforms:

  1. 1. Simple navigation
  2. 2. Clean & readable interface
  3. 3. Functional & efficient
  4. 4. Design consistency

My Role

  • Designing UI components using auto-layout, variables, variant properties, text, and more.

  • Conduct design validation with users to ensure the solution meets their needs.

  • Develop a structured design system including typography, color, and other elements.

  • Create application interfaces and hand them off to developers for implementation.

Project Flow

👩‍💻 Project Scope and Scale

  • 01. Navigation Menu

    This system enables users to purchase courses, access structured learning materials, earn certificates, and participate in community discussions to share experiences and ask questions.

  • 02. Three Different User Roles Views

    To meet diverse operational needs and ensure proper access control:

    1. 1. Admin: Manage platform & users
    2. 2. Student: Learn, earn certificates, and community discussions
    3. 3. Instructurs: Create & manage courses, and community discussions

👤 Use Case

Use cases describe specific scenarios of how each role interacts with the system to achieve particular goals.

🔍 Benchmarking

The system interface design process is guided by benchmarks from well-known educational technology platforms in Indonesia, such as Udemy, Myskill, and Dicoding, to ensure a more intuitive and familiar user experience.
